
# Oppdatering v3.14.8 - Solid Matematikk-arkitektur

Denne oppdateringen re-implementerer hele kjeden for hvordan matematikk blir generert, prosessert og vist.

### Endringer:
*   **Arkitektonisk endring**: `LatexRenderer` splitter ikke lenger innholdet i flere HTML-elementer. Hele tekstblokken sendes nå som én enhet til MathJax, noe som hindrer at matematiske koder blir brutt opp.
*   **Robuste Delimitere**: Instruert Gemini til å bruke `\( ... \)` og `\[ ... \]` som er de mest pålitelige tegnene for matematikk i moderne nettlesere.
*   **Pre-wrap Rendering**: Bruker CSS `white-space: pre-wrap` i stedet for `<p>`-tagger for å bevare linjeskift uten å endre DOM-strukturen.
*   **Forbedret Visuell Flyt**: Komponenter skjules med 0% gjennomsiktighet frem til MathJax bekrefter at rendringen er 100% fullført, noe som fjerner "hopping" og visning av råkode.
