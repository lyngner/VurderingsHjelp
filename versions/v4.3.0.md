
# Oppdatering v4.3.0 - Global Task Navigation

Denne oppdateringen gjør det betydelig raskere å navigere på tvers av mange elevbesvarelser ved å introdusere oppgave-basert filtrering.

### Nye funksjoner:
*   **Globalt Oppgavefilter**: I "Kontroll"-steget kan du nå velge en spesifikk oppgave (f.eks. "2B") for å umiddelbart se hvilke elever som har besvart akkurat denne oppgaven. Dette gjør det enklere å rette én oppgave av gangen.
*   **Forbedret Oppgave-ID i Manual**: Fikset en bug der deler av oppgavebeskrivelsen kunne havne inni det sirkulære ID-merket. Nå viser merket KUN oppgavenummer og bokstav (f.eks. "3B" eller "2").

### Forbedringer:
*   **Strammere Poeng-policy**: KI-en er nå programmert til å konsekvent bruke 2.0 poeng som standard maksverdi per deloppgave.
*   **Renere Kandidatkort**: Kandidatlisten i sidebar er optimalisert for å ta mindre plass og gi bedre oversikt over elevens fremgang.
*   **Bedre OCR-kobling**: Systemet er flinkere til å koble sider mot rettemanualen selv ved uvanlig håndskrift.
