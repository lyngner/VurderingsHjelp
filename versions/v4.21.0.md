
# Oppdatering v4.21.0 - Safety & Technical Enforcement

Denne oppdateringen fokuserer på å låse de tekniske gjennombruddene fra v4.20.0 for å hindre regresjon og degenerasjon av kildekoden.

### Endringer:
*   **Teknisk Grunnlov**: Implementert en ny seksjon i `README.md` som forklarer "Rotate-then-Bisect"-algoritmen i detalj. Dette sikrer at KI-en forstår *hvorfor* Canvas API brukes slik det gjør.
*   **Inline Documentation**: Lagt til kritiske kommentarer i `fileService.ts` som advarer mot å endre rekkefølgen på rotasjon og splitting.
*   **Stabilitetsgaranti**: Ved å dokumentere den "Fysiske Pipelinen", sikrer vi at fremtidige endringer ikke faller tilbake på upålitelige CSS- eller metadata-løsninger.
*   **Forsterket LaTeX-logikk**: Lagt til føringer for hvordan `LatexRenderer` skal håndtere asynkronitet for å fjerne flimmer permanent.

### Teknisk:
*   Sikret at `splitA3Spread` alltid opererer på et ferdig rotert canvas.
*   Oppdatert `README.md` til å fungere som en teknisk master-manual for KI-en.
