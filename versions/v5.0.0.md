
# Oppdatering v5.0.0 - Context-Aware Intelligence

Dette er en stor oppdatering som radikalt forbedrer systemets evne til å gjenkjenne hvilken oppgave en elev svarer på, og fjerner "støy-oppgaver" som 1I og 1III.

### Endringer:
*   **Context-Aware Task Mapping**: KI-en får nå hele rettemanualen som kontekst under bildeanalysen. Den "gjetter" ikke lenger bare på overskrifter, men bruker innholdet i teksten til å forstå hvilken oppgave i whitelisten eleven faktisk svarer på.
*   **Evidence-Based Identification**: Instruert Gemini til å bruke sin egen transkripsjon som bevis. Hvis teksten ligner på beskrivelsen for en oppgave i fasiten, velges denne automatisk.
*   **Full KI-Audit**: Opprettet `MODEL_DOCUMENTATION.md` som detaljerer modeller, strategier og kostnader (ca. 1,- NOK per elev).
*   **Zero-Scroll Sync**: Garantert at hovedvisningen alltid starter på toppen av første side ved kandidatbytte.
*   **Hard Whitelist Reinforcement**: Fjernet muligheten for KI-en til å opprette egne oppgavenavn; alt som ikke matcher fasiten eller er åpenbart, blir nå markert som "UKJENT".

### Teknisk:
*   Oppdatert systeminstrukser i `geminiService.ts` til å inkludere dynamisk kriterie-kontekst.
*   Oppdatert `MODEL_DOCUMENTATION.md` med kostnadsanalyse for v5.0.0.
