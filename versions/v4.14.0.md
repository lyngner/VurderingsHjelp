
# Oppdatering v4.14.0 - Prompt Isolation Fix

Denne oppdateringen løser en kritisk feil der KI-modellens egne instruksjoner dukket opp som transkripsjon i "Kontroll"-steget, spesielt for Word-dokumenter.

### Endringer:
*   **Instruction Blacklist**: Lagt til eksplisitte forbud i `systemInstruction` mot å inkludere instruksjonstekst i `fullText`-feltet.
*   **Schema Descriptions**: Bruker nå `description`-feltet i JSON-skjemaet for å guide modellen til å kun returnere elevens innhold i transkripsjonsfeltene.
*   **Digital Presisjon**: Optimalisert `analyzeTextContent` for å bedre skille mellom dokumentets metadata (topptekst med Kandidatnr) og selve brødteksten.
*   **Feilretting**: Gjenopprettet ryddig LaTeX-visning i transkripsjonsboksen ved å fjerne modellens forklarende støy.

### Teknisk:
*   Oppdatert `geminiService.ts` med strengere `responseSchema`.
*   Sikret at `rawText` fra Mammoth blir behandlet som en ren kilde for KI-analysen.
