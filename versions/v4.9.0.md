# Oppdatering v4.9.0 - Reasoning Isolation & Precision Mapping

Denne oppdateringen løser problemene med at AI-modellens 'tanker' lakk inn i transkripsjonene, og at oppgaver ble feilaktig kategorisert som 'UKJENT'.

### Endringer:
*   **Reasoning Isolation**: Gemini 3 Pro er nå underlagt ultra-strenge instrukser om å ALDRI inkludere interne vurderinger eller estimater i JSON-feltene. Dette fjerner støy som "4B (EST.)" fra brukergrensesnittet.
*   **Precision Task Mapping**: Forbedret logikken som kobler elevbesvarelser mot rettemanualen. Systemet er nå flinkere til å kjenne igjen kombinasjoner som "Oppgave 4b" og mappe dem korrekt selv om AI-en returnerer litt varierende tekstformater.
*   **Sanitert ID-håndtering**: Kandidat-IDer blir nå konsekvent renset for tekststøy (f.eks. "Kandidat 104" blir "104") for å sikre at alle sider samles hos riktig elev umiddelbart.
*   **Forsterket A3-splitting**: Ytterligere instrukser for å sikre at landskapsskanninger alltid vurderes som A3-oppslag og splittes i to vertikale A4-sider.
*   **Robuste Sidebars**: Kandidatlisten i sidebar er nå renset for lange tekststrenger i oppgave-pillene, noe som bevarer det kompakte designet.
